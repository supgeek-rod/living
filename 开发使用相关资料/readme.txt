// 软件功能
	# 提醒事项(100%)	增删改查		// 记录工作中、生活中的零碎任务
	# 备忘录(100%)		增删改查		// 记录一些笔记，较多文本内容
	# 日历(40%)			前台增改		// 安排记录工作、生活事件 
	# 提醒(0％)							// 提醒事项和日历事件的到期提醒 
	# 搜索(0%)							// 搜索相关数据
	# 智能添加(0%)						// 智能添加提醒事项或者日历事件


// 相关技术
	# 基于ThinkPHP框架
	# jQuery
	# bootstrap (前端框架)
	# bootstrap-datetimepicker (日期选择插件)
	# fullcalendar 1.6.4 (日历插件)
	
// 安装使用
	# 创建MySQL数据库,导入数据库文件(默认MySQL [主机:loclhost, 用户:root, 密码:root 库名:living] | 否则需要修改ThinkPHP配置)
	# 测试帐号:	test@living.com 密码: test
	

// 协同开发
	# 在你生活和工作的时候，会有很多繁杂琐碎的任务？经常需要记录一些文本内容？需要安排你的生活和工作，并能记录以便日后查看？
	对，你可以通过apple产品中的[提醒事项][备忘录][日历]等软件，并可在手机、电脑中无缝同步。这是方法简单高效管理工作、生活的好工具。
	如果你是windows用户安卓用户，你用的是什么工具呢？或许你并不需要这个东西，因为你不知识这3个小东西能给你带来怎样的惊喜！
